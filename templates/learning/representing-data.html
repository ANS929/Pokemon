Representing Data

Introduction:

We can display data in many different ways. We will learn about frequency tables, line plots, and bar graphs.

Frequency Tables:

Frequency is the number of times a value occurs. A frequency table lists items and shows the number of times the items occur.

Example: The frequency table below shows the results of asking 20 students to tell their favorite subject. We can make different observations, such as Mathematics was the most popular subject.

Line Plots:

A line plot displays data as points above a number line, showing the frequency of each value.

Example: The line plot below shows the number of students in several different classrooms. We can see that there are 4 classrooms with 29 students.

Bar Graphs:

A bar graph displays bars of different heights. The bars represent a frequency or a number of items for each category.

Example: The bar graph below shows the percentage of sales for each product category in a store. We can observe that Electronics is the third most popular category.
